Timesheets Sheet for Employee Odoo Apps

Employee timesheet sheet odoo apps is use to enter/fill timesheet of employee and record Timesheet Sheet for Employee, this feature has been deprecated from Odoov10 community edition, so here this odoo module provide all kind of same features for community edition of odoo where each employee user can able log his/her timesheet entry, user can submit his/her timesheet to manager for approval. Timesheet manager will get notification for timesheet approval and timesheet user can only have access to his/her timesheet, timesheet manager can able to see/access timesheets of all employeee/users.

Features

Timesheet Sheet to Approve By Manager

Manager can approve or refuse employee timesheet.

Employee Timesheet Report.

Employee Timesheet Report with all timesheet entry details.

Timesheet Analytic Report.

Manager can view analytic report by employee, by project and by task.

Timesheet can review from employee record.

Employee form view you can see timesheet button.

Employee can create and fill Timesheet Sheet for time period. After filling timesheet by employee, Employee can submit timesheet to manager for approve.
